Introduction

Gpp provides Ant tasks supporting general file preprocessing based on the Groovy Template Engine. The core functionality is provided as an Ant filter for use with tasks that supports filter chains. Also included are replacement tasks for <copy> and <javac> that integrate support for preprocessing.

History

Gpp is essentially a Groovy port of VPP, which was originally based on the Velocity template engine.
